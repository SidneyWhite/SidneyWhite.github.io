1. What is the difference between the following 2 statements?
setTimeout(booyah, 2000);
setTimeout(booyah(), 2000);

setTimeout(booyah, 2000); - Executes booyah function after 2000ms.
setTimeout(booyah(), 2000); - Executes booyah function right away and setTimeout does not work.

4. What is "Unobtrusive Javascript"? What is the practical application of Unobtrusive Javascript (and the reasons for using it)?

Unobtrusive JavaScript is a general approach to the use of JavaScript in web pages. Website, and it’s content, should still be accessible when JavaScript is unavailable and the only way to guarantee this is by employing Unobtrusive JavaScript techniques.
Adding JavaScript to your site should be an additive process, bringing better experiences to those who have the capabilities of running your code, not a destructive one for those who for whatever reason have JavaScript disabled or blocked or not fully implemented. This is called Progressive Enhancement.
Simple practical ways would be including JavaScript at the end of your html file. And remove all the JavaScript codes from your html and move them into JavaScript files.
